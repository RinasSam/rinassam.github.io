11 /*
12  * @file
13  * SIFToo low-level driver.
14  *
15  * The SBUS Interface (SIF) has 3 DMA channels, with the first 2 reserved
16  * for SCEI's SIF RPC interface. As far as I know, the last channel is
17  * only used for debugging within the EE's kernel (I haven't seen it in use
18  * on the IOP yet, I guess they decided to leave those modules out of the
19  * BIOS). So on consumer units SIF DMA channel 2 (SIF2) is unused.
20  *
21  * SIF2 is a bidirectional channel, which means data can be transferred in
22  * either direction on the SBUS. The SIFToo driver provides an interface
23  * for IOP drivers to communicate via the EE using this channel.
24  */
