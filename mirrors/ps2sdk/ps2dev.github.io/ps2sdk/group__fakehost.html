Entry point for IRX.

Parameters
argcNumber of arguments.
argvPointer to array of arguments.
Returns
Module Status on Exit.

return values: MODULE_RESIDENT_END if loaded and registered as library. MODULE_NO_RESIDENT_END if just exiting normally.
